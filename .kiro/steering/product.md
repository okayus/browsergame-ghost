# Product Overview

ブラウザゲーム「Ghost」 - ゴーストを捕まえて育成・バトルする2D RPG

## Core Capabilities

- **俯瞰視点マップ探索**: グリッドベースのマップを上下左右に移動してゴーストと遭遇
- **ゴースト捕獲・育成**: 野生のゴーストを捕まえ、レベルアップで強化
- **ターン制コマンドバトル**: 捕まえたゴーストで戦う戦略的バトルシステム
- **タイプ相性システム**: ゴーストの種類による有利・不利の相性

## Target Use Cases

- ブラウザで手軽に遊べるモンスター収集RPG
- ターン制バトルの戦略的なゲーム体験
- 進行状況を保存してプレイを継続

## Value Proposition

- **ブラウザ完結**: インストール不要、どこからでもプレイ可能
- **収集・育成要素**: 多様なゴーストを集めて最強パーティを構築
- **戦略性**: タイプ相性とステータスを考慮したバトル

## Game Elements

### プレイヤー
- マップ上でキャラクターを操作
- 手持ちゴースト（パーティ）を管理
- アイテムを使用（捕獲用、回復用）

### ゴースト
- 種類ごとに固有のタイプ・能力
- レベルと能力値（HP、攻撃、防御、素早さなど）
- 覚える技（わざ）

### バトル
- 野生ゴーストとのエンカウント
- ターン制コマンド選択（たたかう、アイテム、逃げる、捕まえる）
- タイプ相性によるダメージ倍率
